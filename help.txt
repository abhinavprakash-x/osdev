1. Essential Tools:
sudo apt update
sudo apt install build-essential bison flex libgmp3-dev libmpc-dev libmpfr-dev texinfo wget git nasm qemu-system-x86

Cross Compiler Download (elf-gcc):
wget https://ftp.gnu.org/gnu/binutils/binutils-2.41.tar.gz
wget https://ftp.gnu.org/gnu/gcc/gcc-13.2.0/gcc-13.2.0.tar.gz

Extract:
tar -xvf binutils-2.41.tar.gz
tar -xvf gcc-13.2.0.tar.gz

Build:
mkdir build-binutils
mkdir build-gcc

cd ~/osdev/build-binutils
../binutils-2.41/configure \
  --target=i686-elf \
  --prefix="$HOME/osdev/cross" \
  --disable-nls \
  --disable-werror
make -j$(nproc)
make install

cd ~/osdev/build-gcc
../gcc-13.2.0/configure \
  --target=i686-elf \
  --prefix="$HOME/osdev/cross" \
  --disable-nls \
  --enable-languages=c \
  --without-headers
make all-gcc -j$(nproc)
make all-target-libgcc -j$(nproc)

make install-gcc
make install-target-libgcc

Add to Path:
echo 'export PATH="$HOME/osdev/cross/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

2. Folder Structure:
osdev/
│
├── build/              → compiled binaries
├── cross/              → cross compiler toolchain
├── src/
│   ├── bootloader.asm  → bootloader
│   ├── shell.asm       → kernel (real OS logic)
│
└── help.txt            → documentation (this file)
